◆ MAX_SAFE_BATT_VOLTAGE_READ

const float Battery::MAX_SAFE_BATT_VOLTAGE_READ = 2.756
private

The maximum voltage the battery can safely hold.

Calculations:

  • 5-cell lipo * 4.3 v per cell = 21.5 battery volts
  • 68k and 10k ohm voltage divider, analog in read voltage 2.756
Note
Voltages above this threshold indicate that the battery is overvolted!
